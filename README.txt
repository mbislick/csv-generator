## About this
This is a project that contains a simple java application that queries a REST API and 
processes the response's JSON to convert it into a CSV file. It is a possible
solution to the GoEuro's development test problem described in: 
https://github.com/goeuro/dev-test

To compile it use:
mvn clean compile assembly:single

This project also has some simple unit tests that you can run using:
mvn test
